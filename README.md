## Homework module 3

### Переменные и типы

Для начала работы с калькулятором, нам понадобится:

- Создать новый проект
- Объявить пакет main
- Объявить функцию main
- Создать константы конвертации
  - Из USD в EUR
  - Из USD в RUB
  - Рассчитать EUR в RUB на основании первых двух

В git создать ветку: 1-start  
Папка: 1-converter
